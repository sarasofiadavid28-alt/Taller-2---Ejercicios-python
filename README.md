# Python Ejercicios

Este repositorio contiene varios ejercicios hechos en **Python** para practicar los conceptos básicos de programación. Los ejercicios están organizados por secciones y van desde temas simples hasta un pequeño sistema completo al final.

Al inicio hay ejercicios sobre **variables y entrada de datos**. En esta parte se hacen programas que piden información al usuario, como un registro de nombre, edad y ciudad, una calculadora básica, una validación simple de correo electrónico, un verificador de contraseñas seguras y un convertidor de unidades con un menú.

Luego hay ejercicios de **condicionales**, donde el programa toma decisiones dependiendo de los datos que ingresa el usuario. Aquí se incluyen ejemplos como clasificar personas por edad, crear un menú con diferentes opciones, una calculadora que permite hacer varias operaciones, un sistema que convierte notas numéricas a letras y un simulador de descuentos según la categoría de un cliente.

Después vienen ejercicios usando **ciclos (`for` y `while`)**. Estos programas repiten acciones varias veces, como generar números pares hasta un límite, sumar números hasta que el usuario escriba 0, buscar un nombre dentro de una lista, mostrar tablas de multiplicar y eliminar números repetidos de una lista.

También hay ejercicios para practicar **listas y diccionarios**. En esta parte se desarrolla una lista de compras donde se pueden agregar o eliminar productos, un directorio de contactos que guarda nombres y teléfonos, un pequeño gestor de inventario, un analizador que calcula máximo, mínimo, suma y promedio de varios números, y un comparador que muestra qué elementos son iguales o diferentes entre dos listas.

Más adelante se trabajan **funciones**, que sirven para organizar mejor el código. Aquí se crea una función para generar saludos dependiendo de la hora del día, otra para calcular el promedio de una lista, una calculadora donde cada operación está en una función diferente, un detector de palíndromos y una función recursiva para calcular factoriales.

Al final hay un **mini proyecto** que integra todo lo aprendido: un sistema de gestión de biblioteca. Este programa permite agregar libros, mostrarlos, buscarlos por título o autor, prestarlos, devolverlos y eliminarlos. También puede mostrar estadísticas de los libros y guardar la información en un archivo de texto.

El objetivo de este repositorio es practicar programación en Python paso a paso y aplicar los conceptos básicos en ejercicios y en un pequeño sistema completo.
